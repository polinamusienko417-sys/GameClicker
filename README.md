Cyber Hunter 2026
Проста аркадна гра, створена за допомогою Python і Tkinter.
Гравець повинен натискати на рухому ціль, щоб набирати очки та підвищувати рівень, поки не закінчаться життя.

🎮 Геймплей
Натискай на кнопку !, яка рухається по екрану.
За кожні 5 очок:
підвищується рівень,
збільшується швидкість гри,
змінюється колір фону.
Якщо клікнути повз ціль — втрачається життя.
Набери 100 очок, щоб перемогти.

🛠 Можливості
Випадковий рух цілі
Система очок
Рівні та збільшення складності
Система життів
Вікна перемоги та програшу
Простий графічний інтерфейс на Tkinter

📦 Вимоги
Переконайся, що Python встановлений на комп’ютері.

Потрібна бібліотека:

tkinter (зазвичай уже входить до Python)

▶️ Як запустити
Збережи код у файл main.py
Відкрий термінал або командний рядок
Виконай команду:
python main.py

🎯 Керування
ЛКМ по цілі → +1 очко
ЛКМ повз ціль → -1 життя

🏆 Умова перемоги
Набрати 100 очок до того, як закінчаться всі життя.

❌ Умова програшу
Втратити всі 5 життів через промахи.
